# Справка по API

Все сгенерированные API связаны с основным vert.x API, потому для лучшего понимания вам следует ознакомиться с документацией vert.x и скорректировать данные для импорта/примеры, как описано в пункте Функциональная совместимость vert.x.

# Vertx

Основное API Vert.x содержат основу для написания приложений на Vert.x и низкоуровневую поддержку HTTP, TCP, UDP, файловой системы, асинхронной обработки потоков и многих других составных элементов, а также используется многими другими компонентами Vert.x.
