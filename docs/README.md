### As Easy as 1, 2, 3

``` bash
# (1) create
npm init @es4x project

# (2) install dependencies
npm install # OR yarn

# (3) run
npm start # OR yarn start
```

::: warning COMPATIBILITY NOTE
ES4X requires [GraalVM](https://www.graalvm.org) or Java >= 8.
:::
