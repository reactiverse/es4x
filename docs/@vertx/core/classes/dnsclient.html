lookup

  • lookup(name: string, handler: function | Handler<AsyncResult<string>>): DnsClient
    • Defined in index.d.ts:2117

    Try to lookup the A (ipv4) or AAAA (ipv6) record for the given name. The first found will be used.

    Parameters

    • name: string

      the name to resolve

    • handler: function | Handler<AsyncResult<string>>

      the {@link io.vertx.core.Handler} to notify with the {@link io.vertx.core.AsyncResult}. The handler will get notified with the resolved address if a record was found. If non was found it will get notifed with {@code null}. If an error accours it will get failed.

    Returns DnsClient

    a reference to this, so the API can be used fluently
