This error handler will be pass to all StompClientConnection created from this client. Obviously, the client can override it when the connection is established.

receivedFrameHandler

  • receivedFrameHandler(handler: function): StompClient
    • Defined in index.d.ts:234

    Configures a received handler that gets notified when a STOMP frame is received by the client. This handler can be used for logging, debugging or ad-hoc behavior. The frame can still be modified at the time.

    When a connection is created, the handler is used as {@link StompClientConnection#receivedFrameHandler(Handler)}.
