register

  • register(name: string, procedure: function): HealthChecks
  • register(name: string, timeout: number, procedure: function): HealthChecks
    • Defined in index.d.ts:129

    Registers a health check procedure.

    The procedure is a {@link Handler} taking a {@link Future} of Status as parameter. Procedures are asynchronous, and must complete or fail the given {@link Future}. If the future object is failed, the procedure outcome is considered as DOWN. If the future is completed without any object, the procedure outcome is considered as UP. If the future is completed with a (not-null) Status, the procedure outcome is the received status.

    This method uses a 1s timeout. Use {@link #register(String, long, Handler)} to configure the timeout.

    Parameters

    • name: string

      the name of the procedure, must not be {@code null} or empty

    • procedure: function

      the procedure, must not be {@code null}
