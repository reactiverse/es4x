# API Reference

Całe wygenerowane API odnosi się do vert.x core API, więć aby lepiej zrozumieć dane zagadnienie powinieneś przeczytać manuale vert.x i oswoić się z importami/przykładami wyjaśnionymi w vert.x interop.

# Vertx

Vert.x core API jest podstawą dla pisania programów przy użyciu Vert.x i niskopoziomowego wsparcia dla HTTP, TCP, UDP, plików systemowych, asynchronicznych strumieni i innych bloków. API używane jest również przez inne komponenty Vert.x.
