ES dla Eclipse Vert.x

Nowoczesny runtime JavaScript dla Eclipse Vert.x

Get Started →

Po pierwsze prostota

Minimalny setup ze strukturą projektu skoncentrowaną wokół npm pozwala na skupieniu się na tworzonym kodzie.

Napędzany Vert.x

Ciesz się skalowalnością i wydajnością Vert.x, korzystaj z komponentów Reactive Vert.x w JavaScript i rozwijaj je za pomocą JS lub TypeScript.

Wydajność

ES4X działa na bazie GraalVM oferując świetną wydajność dla aplikacji JavaScript zbliżoną do tej oferowanej przez Javę, a nawet i lepszą.

# Łatwe jak 1, 2, 3

# (1) create
npm init @es4x project

# (2) install dependencies
npm install # OR yarn

# (3) run
npm start # OR yarn start

NOTA KOMPATYBILNOŚCI

ES4X wymaga GraalVM lub Javy >= 8. Jeśli nie jesteś pewien, która wersja jest zainstalowana na Twoim komputerze rozważ użycie jabba.
