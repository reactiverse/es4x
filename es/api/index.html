# Referencia API

Todos los APIs generados estan relacionados con los APIs de vert.x principales, asi que para entenderlo mejor deberias leer los manuals de vert.x y adaptar el manual imports/examples como se explica en vert.x interop.

# Vertx

Los APIs en el nucleo de Vert.x contienen la estructura para escribir aplicaciones Vert.x y suporte de bajo nivel para HTTP, TCP, UDP, sistema de ficheros, flujos asincronicos y muchos otras piezas fundamentales. Tambies lo utilizan muchos componentes de Vert.x
